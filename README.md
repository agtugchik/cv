# Artur Shvedenko CV

## Quick start sequence

1. Install dependencies

```
npm i
```

2. Run a locacal development server

```
npm run dev
```

## Commands

- Run a local development server

```
npm run dev
```

- Build the project

```
npm run build
```

- Run a local development server that serves the build solution from `./dist` after build command

```
npm run preview
```

- Run Eslint

```
npm run lint
```

- Run Prettier

```
npm run format
```
